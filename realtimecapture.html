**RealSynth RealTimeCapture Plugin** **Video and Image Sequence Capturing** RealSynth RealTimeCapture Plugin is part of [RealSynth Simulacron](index.html). This plugin is still in Beta. RealTimeCapture =============================================================== FFmpeg is used to write out image sequences or videos to disk. * FFmpeg supports a wide variety of output formats * Encoding happens in a separate process Prerequisites --------------------------------------------------------------- A recent FFmpeg build (e.g 20170915-6743351) needs to be present on the machine.
